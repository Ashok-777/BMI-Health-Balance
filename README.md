## ğŸ§ª How It Works

1. Enter your **weight and height** (toggle Metric / Imperial units).  
2. Set your **age** using:
   - Date of Birth (calendar), or  
   - Manual age input  
3. Click **Calculate** to instantly view:
   - BMI value  
   - WHO health classification  
4. Click **Save** to store the result locally.  
5. Open the **Statistics tab** to view BMI & weight trends.  
6. Toggle **Dark / Light Mode** using the sun/moon icon.  

> The statistics use a **7-entry rolling window** to clearly show recent health trends.
